# Restaurant Order Management System

A desktop-based Restaurant Order Management System built with Python, PyQt, and SQLite.  
This application allows restaurants to manage orders, products, tables, and ticket printing through an intuitive graphical user interface.

---

## Features

- Create, update, and close orders
- Change order status in real time
- Print order tickets
- Manage menu items and tables
- View order history
- Daily overview with calendar insights
